## How to contribute to ItsOurMission.org
The following is a set of guidelines for contributing to ItsOurMission.org, which are hosted in the np-tech on GitHub. These are mostly guidelines, not rules. Use your best judgment, and feel free to propose changes to this document in a pull request.

#### Table Of Contents
01. [Did you find a bug](#did-you-find-a-bug)
02. [Want to add your product/service to the website](want-to-add-a-product-service-to-the-website)
03. [Code of Conduct](#code-of-conduct)


#### **Did you find a bug?**
* **Ensure the bug was not already reported** by searching on GitHub under [Issues](https://github.com/np-tech/np-tech.github.io/issues).


#### **Want to add a product/service to the website?**
* Make sure that your product/service is for nonprofits
* Fork the repo, add your product/service and create a pull-request
* **This is strickly for nonprofits. If the product/service is free for them to use, even better**

## Code of Conduct
This project and everyone participating in it is governed by the [ItsOurMission.org](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code. Please report unacceptable behavior to [richardchiriboga@gmail.com](mailto:richardchiriboga@gmail.com).
